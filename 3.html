* JS
- 사용자와 상호작용을 하는 언어
- HTML을 제어하는 언어
- 웹페이지를 동적으로 만들어줌
- HTML은 1+1을 그대로 '1+1'로 표시하지만 JS는 2로 표현함 즉, 유동적임
-

* document
- document.querySelector('abc') : 태그 이름이 abc인 태그를 선택
- document.querySelector('#abc') : 태그 id가 abc인 태그를 선택
- document.querySelector('#abc').value : 태그 id가 abc인 태그의 value값을 가져옴
- .write : 화면에 표시
- document.querySelectorAll('a') : 태그 이름이 a인 태그 모두 선택

* 이벤트
- onclick : 클릭했을 때 "..."안의 JS코드를 실행
- onchange : 변화가 있을 때 "..."안의 JS코드를 실행
- onkeydown : 키가 눌러졌을 때 "..."안의 JS코드를 실행
- onmouseover : 마우스를 범위 안에 올렸을 때 "..."안의 JS 코드를 실행
- onmouseout : 마우스가 범위 밖에 나갔을 때 "..."안의 JS 코드를 실행
- onload : 브라우저 로딩이 끝났을 때 "..."안의 JS 코드를 실행

* 명령어
- 개발자모드에서 console창에서 JS 가능 (현재 웹페이지가 대상)
- "", '' 모두 쓰임
- 'abc'.length : 문자의 길이
- 'abc'.toUpperCase() : 대문자로 변경
- 'abc'.indexOf('a') : 문자 abc에서 a의 위치
- 'abc'.trim() : 문자열의 공백 제거
- "abc" + "de" : 문자열 합치기

- boolean

* 변수
- var x=1; var y=1; 처럼 대입연산자를 이용해서 변수 넣을 수 있음

* 리팩토링
- 코드를 좀 더 효율적으로 변경하여 유지보수를 용이하게 만듬
- 자기 자신을 가리키는 코드는 this. 로 간단히 표현할 수 있음
- 'var target = document.querySelector('body');'
- alist[i] 처럼 배열로 선언 가능
- alist[i].style.color = 'blue' 처럼 변경 가능

* 매개변수, 인자
- function sum(a, b) {...} sum(2,3); 라고 있을때,
- a,b는 매개변수(parameter) 2,3은 인자(argument)라고 한다.

* 객체
