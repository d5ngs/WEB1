WEB과 민영이

==========================================

* 인터넷이 동작하기에는 최소 2대의 컴퓨터가 있어야 한다.



Web Server가 설치된 컴퓨터에는

  1. info.cern.ch라는 주소를 부여
  2. index.html이라는 파일을 지정
  3. 서버 라고 할 수 있고, 클라이언트에게 response함
  4. index.html을 찾고 내용을 읽어서 클라이언트에게 신호를 보냄



Web Browser가 설치된 컴퓨터에는

  1. http://info.cern.ch/index.html 이라는 주소를 입력할 수 있음
  2. 클라이언트 라고 할 수 있고, 서버에게 request함
  3. 서버로부터 신호가 왔을 때, 웹브라우저는 신호를 읽고 웹페이즈를 화면에 출력



* 우리는 늘 웹브라우저와 익숙하게 지내고 있다.
* 이번에는 웹서버로 눈을 돌려보자. 익숙해지는 것이다.
